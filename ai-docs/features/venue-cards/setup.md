# Setup - Venue Cards

## Install
```bash
# Add shadcn/ui card component (if not already added)
npx shadcn@latest add card

# Carousel already added from onboarding-carousel
```

## Config
No additional configuration needed.

## Run
```bash
npm run dev
```

## Test
```bash
# Run component tests
npm run test -- --testPathPattern=venue-cards

# Run E2E interaction tests
npm run test:e2e -- --spec=venue-cards
```
