# Setup - Onboarding Carousel

## Install
```bash
# Add shadcn/ui carousel component
npx shadcn@latest add carousel
```

## Config
Carousel component uses Embla Carousel under the hood.
No additional configuration needed.

## Run
```bash
npm run dev
```

## Test
```bash
# Run component tests
npm run test -- --testPathPattern=onboarding-carousel

# Run E2E navigation tests
npm run test:e2e -- --spec=onboarding-carousel
```
